# Web GIS Project

This project contains web-based Geographic Information System (GIS) visualizations for forest area data across different years.

## Structure

- `index.html`: Main entry point
- `analisis.html`: Analysis page
- `perbandingan.html`: Comparison page
- `sebaran.html`: Distribution page
- `GIS/`: General GIS folder with OpenLayers implementation
- `luas20XX/`: Year-specific folders (2020-2024) with Leaflet-based maps
- `luashutan/`: Forest area data folder

## Technologies Used

- HTML/CSS/JavaScript
- OpenLayers (in GIS folder)
- Leaflet (in year-specific folders)
- QGIS2Web exported maps

## Usage

Open the respective HTML files in a web browser to view the maps and data visualizations.
